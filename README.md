# Oladipupo Store Data Analysis (Interactive Dashboard creation )
## Project Objective
Oladipupo Store wants to create an annual report of Route Profitablility for 2022, 2023 and 2024, to evaluate it's profitable routes and profits made from the routes.
## Dataset used
https://docs.google.com/spreadsheets/d/1CAT5kcz5HGAEOz4lmiokEmLFbbs6cQQI/edit?usp=drive_link&ouid=104177868748180292987&rtpof=true&sd=true
## Process
- Verify data for any missing values and anomalies, and sort out the same .
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created tables and charts according to the questions asked.
- Merge all  tables into one dashboard and apply slicer to make dynamic.
## KPIs
KPI's
- Total Profit
- Revenue per mile
- Cost per mile
- Profit Margin

  ## Project Insight  
-The dashboard shows a very strong overall profit of 197.30M.
-Revenue per mile (2.44) is significantly higher than cost per mile (0.83), indicating efficient operations.
-Cities like Seattle, Charlotte, and Columbus appear to contribute the highest profits.
-Destination states such as OR, CA, and WA generate a large share of total revenue.
## Conclusions
-The business is highly profitable with a healthy profit margin above 66%.
-Route operations are cost-efficient since revenue per mile greatly exceeds cost per mile.
-Focusing on top-performing cities and destination states could further increase profitability.
